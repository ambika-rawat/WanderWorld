WanderWorld – Your Complete Travel Companion

Explore the globe, plan smarter, and travel better — all from one beautifully designed website.


📌 Project Overview
WanderWorld is a responsive travel information website built as part of the FWD (Front-End Web Development) project. It provides structured, clear, and visually engaging information about travel destinations across the world — solving the common problem of scattered travel information by bringing everything into one platform.
The website targets students, young adults, first-time travellers, and budget explorers who need a single, reliable place to research and plan their trips.

🎯 Problem Statement
Finding reliable and well-organised travel information across different countries is often difficult. Travellers usually visit multiple websites to understand places, culture, attractions, and travel basics — making the process time-consuming and confusing.
WanderWorld addresses this by offering:

Structured destination information in one place
Interactive tools for planning and budgeting
Simple navigation suitable for all user types
Full responsiveness across mobile, tablet, and desktop

🛠️ Interactive Travel Tools
💱 Currency Converter

Converts between 40+ world currencies
Swap button to reverse conversion instantly
Quick reference table showing 9 common amounts at once
Approximate rates for travel planning

🌐 Phrase Translator

Essential travel phrases in 20+ languages
Languages include Hindi, Japanese, Arabic, Korean, Thai, Turkish, French, Spanish, and more
One-click phrase buttons: Hello, Thank You, How Much?, Help Me, I'm Vegetarian, Where is Toilet?, Water Please
Swap language direction button

🗺️ Trip Planner & Budget Calculator
A 4-step interactive trip planner:

Destination Search — Search and select from the destination database
Dates & Budget — Choose trip duration (3–21 days) and set daily sliders for hotel, food, and activities
Food & Travel Style — Select dietary preference (All / Vegetarian / Vegan / Halal) and travel style (Culture, Adventure, Food Tours, Relaxation, Shopping, Nature)
Generated Itinerary — Full day-by-day schedule with times, activities, and costs + animated budget breakdown bar chart with grand total

🎯 Dart Game

Interactive world map drawn on HTML5 Canvas
Click anywhere on the map to get the nearest destination
Throw Dart button for a fully random destination pick
Shows destination name, continent, best travel time, and estimated flight price
One-click bridge to Trip Planner with the selected destination pre-filled

🥗 Food Preference Filters

Green dot = Vegetarian friendly
Purple dot = Vegan options available
Red dot = Non-vegetarian cuisine
Filter chips in the search bar to browse only matching destinations

🔍 Search & Filter System

Live search across destination name, country, description, and tags
Continent filter: All / Asia / Europe / Americas / Africa / Oceania / Middle East
Food preference filter chips
"No results found" state when no destinations match

📋 Destination Modal
Each destination card opens a detailed modal with:

Currency and language information
Best time to visit and climate
Estimated ticket prices (flights, entry fees, transport)
Food options breakdown (Vegetarian / Vegan / Non-Veg notes)
Must-see attractions list
Local insider tips 

🔮 Future Scope
If this project were to be extended further, the following features could be added:

Google Translate API for full phrase translation support

Map integration using Leaflet.js or Google Maps API
Review and rating system for destinations

More destinations — target 100+ across all continents
